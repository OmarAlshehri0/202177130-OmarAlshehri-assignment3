# Personal Portfolio – Assignment 3

## Project Description
This project is my personal portfolio website for Assignment 3. It builds on previous Assignments by adding more advanced functionality using HTML, CSS, and JavaScript.

The website includes:
- About section
- Projects section
- Skills section
- Design section using an external API
- Contact form with validation
- Dark/light theme 
- Project search, filter, and sort features

The goal of this assignment was to improve the website by adding more stronger logic, API integration, improve user experience, and cleaner documentation.

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Pexels API
- GitHub

## Main Features
- Responsive portfolio layout
- Dark/light mode with saved state
- Project search feature
- Project category filter
- Project sorting by title
- Skills slider
- Design images loaded from API
- Contact form validation with feedback messages
